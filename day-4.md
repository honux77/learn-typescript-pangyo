# TS 4일차

## tsc 설치

```
npm i -g typescript
```

## 컴파일

```
tsc index.ts
```

## ts 설정파일

tsconfig.json을 만들자

```
{
    "compilerOptions": {
        "allowJs": true,
        "checkJs": true,
        "noImplicitAny": true
    }
}```
